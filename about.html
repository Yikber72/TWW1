About This Project

Welcome to The Web's Wardrobe: A Journey Through Design Eras! This website is a submission for the first hackathon hosted at our school. We developed this website to explore the diverse styles expressed throughout the era of the internet. This is a WIP. We look forward to add more content to this website and fill in the gaps created.

What You'll Find Here

Each dedicated era page on this site showcases:

  • Key Design Characteristics: A breakdown of the defining visual and functional elements of that period.
  • Iconic Websites: Screenshots and links (often via the Internet Archive's Wayback Machine) to famous or representative websites from the era.
  • Live Style Snippets: Interactive sections built with modern HTML and CSS to emulate the look and feel of that specific design era, demonstrating some concepts in action.

Acknowledgements

A special thanks to the Internet Archive's Wayback Machine, an invaluable resource that made it possible to revisit and showcase many of the historic websites featured.
